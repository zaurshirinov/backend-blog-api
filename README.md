# 📝 Backend Blog API

Django Rest Framework ilə hazırlanmış RESTful Blog API. Bu API vasitəsilə postlar yaratmaq, siyahılamaq, filtr etmək və şərh əlavə etmək mümkündür.

---

## 🚀 Features

- 📌 Post list və create
- 🔍 Search (title, content)
- 🗂 Filter (category, author)
- ↕️ Ordering
- 💬 Comment sistemi
- 📄 Pagination
- 🔐 Authentication (token/session)

---

## 🛠 Tech Stack

- Python
- Django
- Django Rest Framework
- SQLite (default)
